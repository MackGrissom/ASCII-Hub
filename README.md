## Features

- **Customizable colors** — 8 color presets for single-color animations
- **Adjustable size** — 10px to 18px font size
- **Custom characters** — Type your own characters into text-based animations
- **Tie-dye controls** — 5 palettes, twist & rings sliders
- **Copy JSX** — One click to copy the component with your current settings
- **Zero dependencies** — Pure React, no animation libraries

## Getting Started

```bash
npm install
npm run dev
```

## Deploy to Vercel

```bash
npx vercel
```

Or connect the GitHub repo to [vercel.com](https://vercel.com) for automatic deployments.

## Usage

Each animation is a standalone React component:

```jsx
import { MatrixRain } from './App'

<MatrixRain color="#4ade80" size={14} customChars="HELLO" />
```
